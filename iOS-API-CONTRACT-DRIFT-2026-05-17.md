## Method

- iOS paths: `grep -rohE '"/api/[a-z0-9/_.-]+'` across `ro-control-ios/ROControl`
  and `ro-tools-ios/ROTools`, deduplicated.

## Sunset proposals (file as queue items with BackEndAgent)

1. **`/api/checklists/instances/:id/finalize`** — promote the legacy
   `/complete` to the canonical endpoint, retire `/finalize`. Or commit
   to `/finalize` so RC iOS can drop its fallback. Either way, pick one.
2. **`/api/messaging/channels/:id/messages` (POST)** — pick a sunset date
   for the legacy shape so RT iOS can drop `legacyManagerChannelId` +
   `normalizedPostPlan` (≈40 LOC across `APIService.swift`).
3. **`rocontrol://` custom scheme** — once every install is on a
   Universal-Link-capable build, drop the `Info.plist`
   `CFBundleURLSchemes: [rocontrol]` entry and the `scheme == "rocontrol"`
   branch in `AuthManager.handleIncomingURL`.
